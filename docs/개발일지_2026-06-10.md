### 3-2. 데이터베이스 테이블 설계 및 생성

Supabase Management API를 통해 직접 테이블을 생성하였다.

#### posts 테이블
```sql
create table posts (
  id          bigint generated by default as identity primary key,
  user_id     uuid references auth.users on delete cascade not null,
  title       text not null,
  content     text not null,
  author_name text,
  category    text not null default 'free'
              check (category in ('notice', 'free', 'qna')),
  views       integer default 0,
  created_at  timestamptz default now(),
  updated_at  timestamptz default now()
);
```

#### comments 테이블
```sql
create table comments (
  id          bigint generated by default as identity primary key,
  post_id     bigint references posts on delete cascade not null,
  user_id     uuid references auth.users on delete cascade not null,
  content     text not null,
  author_name text,
  created_at  timestamptz default now()
);
```

#### RLS (Row Level Security) 정책
- 게시글/댓글 **조회**: 모든 사용자
- 게시글/댓글 **작성**: 로그인 사용자만
- 게시글/댓글 **수정·삭제**: 본인 작성글만

#### increment_views 함수
```sql
create or replace function increment_views(post_id bigint)
returns void language sql security definer as $$
  update posts set views = views + 1 where id = post_id;
$$;
```

### 3-4. 게시판(Board) 구현

게시판은 **공지사항 / 자유게시판 / Q&A** 3개 카테고리로 구성하였다.

#### 생성 파일

| 파일 | 역할 |
|------|------|
| `src/pages/board/BoardList.jsx` | 카테고리별 게시글 목록 (페이지네이션) |
| `src/pages/board/BoardDetail.jsx` | 게시글 상세보기 + 댓글 |
| `src/pages/board/BoardWrite.jsx` | 게시글 작성 |
| `src/pages/board/BoardEdit.jsx` | 게시글 수정 |

#### 주요 기능
- 카테고리 탭 전환 (공지사항 / 자유게시판 / Q&A)
- 페이지당 15개 게시글, 페이지네이션
- 조회수 자동 증가 (RPC 함수)
- 댓글 수 목록에서 표시 (`[n]`)
- 공지사항 게시글에 **공지** 뱃지 표시
- 본인 글/댓글만 수정·삭제 가능
- 비로그인 시 로그인 유도 안내


## 5. 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| Kakao Client Secret 없음 | 일반 앱은 비즈니스 인증 없이 Client Secret 미제공 | Kakao는 Client Secret 미설정 시 서버에서 검증 안 함 → dummy 값으로 Supabase 설정 |
| 라우트 충돌 (`/board/write` vs `/board/:id`) | React Router 동적 라우트 우선순위 | 정적 라우트(`write`, `edit/:id`)를 동적 라우트(`:id`) 앞에 배치 |
| HashRouter + OAuth 콜백 | GitHub Pages 정적 호스팅 특성 | `redirectTo`를 origin 기반으로 설정, Supabase PKCE 자동 처리 |
